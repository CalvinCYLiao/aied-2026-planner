# AIED 2026 個人大會日程與旅行規劃器

這是一個為您量身打造的互動式單網頁應用程式 (Single-Page App)，用於管理您在 **AIED 2026 研討會** 期間的個人日程、機票航班、飯店住宿及首爾旅行準備事項。

## 📂 專案檔案結構
* `index.html`：網頁主結構 (Hero 區塊、倒數計時、日程表格、交通及指南面板)
* `style.css`：精美的磨砂玻璃 (Glassmorphism) 深色科技感樣式與響應式排版
* `app.js`：包含 155 個大會完整 Session 資料，以及日程篩選、星號標記、待辦清單和 `localStorage` 瀏覽器快取儲存邏輯。

## 🚀 如何使用
1. 直接在您的電腦上雙擊打開 `index.html` 檔案（使用 Chrome, Safari, Edge 或任何瀏覽器皆可）。
2. **日程規劃器**：
   - 您可以切換「週一」至「週五」的標籤，查看當天所有的 Sessions。
   - 點擊左側的「⭐ 顆星」即可將該 Session 儲存至您的個人專屬日程中。
   - 勾選右上角的「只顯示我標記的日程」即可一鍵匯出您的個人專屬日程。
   - 支援依類別（AIED, EDM, L@S 等）進行篩選。
3. **交通與機票**：
   - 已預先為您填妥去程 BR2148（韓亞航空實際執飛）與回程 BR169 航班明細、出發/抵達時間及航廈、行李限重 (Checked: 23kg, Carry-on: 7kg)。
   - 您可以直接在網頁下方的表單中輸入或編輯您的飯店資訊，點擊儲存後資料將會自動保存在您的瀏覽器中。
4. **準備清單**：
   - 提供一份專門為本次首爾學術會議設計的 Checklist。完成項目後勾選，上方進度條將會動態更新。
5. **首爾旅遊指南**：
   - 提供 COEX Center 地鐵交通指南（2號線三成站 / 9號線奉恩寺站）。
   - 推薦韓國地圖與翻譯 App 實用指南。
   - 整理常用旅行韓語對照表。

*提示：此規劃器完全在您的本地瀏覽器運行，所有新增、修改的資料（如標記的日程、飯店資訊、Checked狀態）均會透過瀏覽器的 `localStorage` 進行儲存，重新整理網頁或關閉後再次開啟，資料也不會遺失！*
